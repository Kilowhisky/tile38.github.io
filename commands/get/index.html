GET key id [OBJECT|POINT|BOUNDS|(HASH geohash)]

Get the object of an id. The default output format is a GeoJSON object.

Output formats:

  • OBJECT: A GeoJSON object.
  • POINT: A standard latitude, longitude point.
  • BOUNDS: A minimum bounding rectangle.
  • HASH: A Geohash.

A Geohash requires a precision of 1 to 22.

Examples

GET fleet truck1 # get geojson object 'truck1' from 'fleet' GET fleet truck1 OBJECT # same as above GET fleet truck1 POINT # get latitude, longitude point GET fleet truck1 BOUNDS # get bounding rectangle GET fleet truck1 HASH 5 # get geohash with precision of 5
