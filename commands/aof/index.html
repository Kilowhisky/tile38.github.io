AOF pos

Related commands
  • AOF
  • AOFMD5
  • AOFSHRINK
  • FOLLOW

Downloads the AOF starting from pos and keeps the connection alive. All data that is written to the AOF will also be written to the connection. This command is used for leader-follower replication.
